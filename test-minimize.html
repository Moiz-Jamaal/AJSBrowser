How it works:

The browser checks for a cookie named BrowserAllowMinimize:

  • If cookie value = "true": Window can be minimized, repositioned, and is not always on top
  • If cookie is missing or = "false": Window stays maximized and always on top

The browser checks every 2 seconds and reapplies restrictions if needed.
